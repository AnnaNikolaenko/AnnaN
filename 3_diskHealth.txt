3. Как можно протестировать Диск Здоровья 
Тест-план (загляни в лекцию если забыл(а))
https://testengineer.ru/chto-takoe-test-plan-i-kak-ego-napisat/
Тест-кейсы (15 и более)
 
Оценка = 10p.

https://www.sportmaster.ru/product/24986420299/
Тренировочный диск от Demix поможет развить координации движений и разогреть мышцы перед тренировкой. Занятия на диске способствуют 
развитию вестибулярного аппарата, растяжке мышц живота и спины, позволяют развить гибкость и укрепить связки и сухожилия голеностопного 
и коленного суставов. Массажная поверхность диска стимулирует кровообращение. 

Тест кейс №1 - Проверка наличия документов и инструкции
Шаги
-Открыть коробку
-Достать все содержимое
Ожидаемый результат 
Вся необходимая документация приложена к данному товару.

Тест кейс №2 - Осмотр тренажера на наличие дефектов
Шаги
-Берем Диск и ставим на ровную поверхность
-Сначала на одну сторону
-Потом на другую сторону
Ожидаемый результат 
Поверхности с обеих сторон ровные и не шатаются.

Тест кейс №3 - Проверка руками сторону Диска с "полосками"
Шаги
-Ставим Диск на ровную поверхность стороной с "полосками" вверх
-Крутим рукой по часовой стрелке
-Крутим рукой против часовой стрелки
Ожидаемый результат 
Диск крутится без задержек, без лишних звуков.

Тест кейс №4 - Проверка руками сторону Диска с "шариками"
Шаги
-Ставим Диск на ровную поверхность стороной с "шариками" вверх
-Крутим рукой по часовой стрелке
-Крутим рукой против часовой стрелки
Ожидаемый результат 
Диск крутится без задержек, без лишних звуков.

Тест кейс №5 - Проверка работы Диск ребенком дошкольного возраста
Шаги
-Берем Диск и ставим на ровную твердую поверхность
-Ставим ребенка на Диск
Ожидаемый результат 
Диск начинает вращаться по и против часовой стрелки при любом движении ребенка, ребенок
устоять не может.

Тест кейс №6 - Проверка работы Диска при массе тела меньше 100кг
Шаги
-Берем Диск и ставим на ровную твердую поверхность
-Встаем на Диск
-Совершаем поворот вправо
-Совершаем поворот влево
Ожидаемый результат 
Диск крутится без проблем и нареканий, свою рабочую функцию выполняет.

Тест кейс №7 - Проверка работы Диска при массе тела больше 100кг
Шаги
-Берем Диск и ставим на ровную твердую поверхность
-Встаем на Диск
-Совершаем поворот вправо
-Совершаем поворот влево
Ожидаемый результат 
Диск практически не крутится, слышны звуки "скрипящие", возможна поломка.

Тест кейс №8 - Проверка наличия QR кода системы госмаркировки
Шаги
-Берем упаковку товара
-Находим QR код
-Скнируем его в приложении Честный Знак
Ожидаемый результат 
Товар найден в системе госмаркировки.

Тест кейс №9 - Проверка габаритов Диска
Шаги
-С помощью линейки или рулетки измеряем диаметр Диска
-Сверяем фактические результаты с заявленными производителем
Ожидаемый результат 
Полученный результат совпадает с заявленным производителем.

Тест кейс №10 - Вращение на Диске на только левой/правой ноге
Шаги
-Берем Диск и ставим на ровную твердую поверхность
-Встаем на Диск
-Поочередно встаем на левую,потом на правую ногу
-Совершаем поворот вправо
-Совершаем поворот влево
Ожидаемый результат 
Выполнять должен подготовленный, тренированный человек.

Тест кейс №11 - Проверка на наличие запаха дешевого пластика
Шаги
-Открываем коробку
-Не доставая товара из упаковки нюхаем содержимое
Ожидаемый результат 
Неприятные запахи отсутствуют.

Тест кейс №12 - Проверка вращения на Диске сидя на корточках
Шаги
-Берем Диск и ставим на ровную твердую поверхность
-Встаем на Диск и садимся на корточки
-Совершаем поворот вправо
-Совершаем поворот влево
Ожидаемый результат 
Упражнения удалось выполнить.

Тест кейс №13 - Проверка вращения Диска на мокрой поверхности
Шаги
-Берем Диск и ставим на ровную твердую поверхность, на которую налили воду
-встаем на Диск
-Совершаем поворот вправо
-Совершаем поворот влево
Ожидаемый результат 
Диск не устойчив, при любых движениях начинает двигаться по поверхности.

Тест кейс №14 - Проверка устойчивости Диска на ровной поверхности
Шаги
-Берем Диск в руки
-Ставим Диск на ровную 
Ожидаемый результат 
Диск стоит ровно, устойчиво.

Тест кейс №15 - Проверка устойчивости Диска на ребре
Шаги
-Берем Диск в руки
-Поставить Диск на ребро
Ожидаемый результат 
Диск падает, возможна поломка.
