Провести тестирование программы hardload.exe, которая позволяет копировать, выбранные в списке файлы в указанный каталог.
Эта программа была специально написана для цели - наглядно показать студентам курса по Тестированию, как происходит работа с программой.
Тестирование проводится по методу "чёрного ящика". Код не тестируется. Тестируем интерфейс (внешний вид) и функционал (работоспособность) программы.
По заданию необходимо составить 15 и более тестовых сценариев, указав фактический результат для каждого тестового сценария, совершить тестовый прогон.
Если будут выявлены дефекты в работе приложения, то необходимо внести их в YouTrack и описать каждый.
А так же составить пользовательскую документацию.
Для тестирования были составлены следующие тестовые сценарии:
1)копирование файла с внешнего носителя
2)Вместо выбора папки, куда писать файл, выбираем сам файл 
3)Копирование одного файла большого размера
4)Копирование сразу нескольких файлов большого размера
5)Копирование файла на внешний носитель
6)Копирование файла (с одним наименованием) несколько раз в одну папку
7)Копирование файла без выбора отметки чек-бокса
8)Копирование файла путем перетаскивания в приложение
9)Копирование файла с разным расширением
10)Копирование файла в скрытую папку
11)Добавление нескольких файлов при помощи клавиши "ctrl"
12)Копирование файла
13)Проверка кнопки "удалить выделенные файлы из списка"
14)Проверка кнопки "остановить грузить"
15)Проверка кнопки "выбор папки, куда писать файлы" 
Срок выполнения тестирования с 04.02.23 по 11.02.23
Для выполнения тестирования понадобиться компьютер, созданные файлы, необходимые для проверки работоспособности программы, знания и умение работать в testrail.
